# Marmitex da Idi 🍽️

Site institucional do **Marmitex da Idi** — restaurante de self service à vontade localizado na Vila Regente Feijó, São Paulo - SP.

## 🧭 Sobre

O Marmitex da Idi nasceu pra deixar o almoço leve e gostoso: buffet bem cuidado, comida fresca e aquele clima que dá vontade de voltar.

- **Self service à vontade** — monte seu prato e repita com tranquilidade
- **iFood** — peça pelo app e receba com praticidade
- **Retirada** — atendimento direto para combinar e retirar

## 📍 Localização

**Endereço:** Rua Júlio Alfredo Manuel, 41 - Vila Reg. Feijó, São Paulo - SP, 03336-030

**Horário:** Segunda à Sábado • 11:00–15:00

## 🚀 Tecnologias

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/)
- [Netlify](https://www.netlify.com/) — deploy e hospedagem

## 💻 Desenvolvimento

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build de produção
npm run build

# Preview do build
npm run preview
```

## 🌐 Deploy

O deploy é feito automaticamente via Netlify ao fazer push na branch principal.
